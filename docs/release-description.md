# Goal Tracking Progress Bar

Goal Tracking Progress Bar adds a compact progress dashboard to Anki's home screen.

Features:

- weekly, monthly, and yearly goals
- optional goals enabled independently
- support for review count, new cards learned, and study time in minutes
- compact progress bars designed to feel at home in Anki's deck browser

Configuration:

- choose which periods are enabled
- choose the metric for each period
- set a target value for each goal
