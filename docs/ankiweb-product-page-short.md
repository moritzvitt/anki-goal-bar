# Goal Tracking Progress Bar

Stay on track with your study goals right from Anki's home screen. 🎯

Goal Tracking Progress Bar adds compact deck-based and custom goal bars so you can instantly see whether you're moving toward your targets or falling behind.

</p>

## Why You Might Want It

- See your progress at a glance 👀
- Stay motivated with visible goals every time you open Anki 🚀
- Track different decks separately or add custom goal windows 📚
- Use goals based on reviews, new cards, or study time ⏱️
- Build streaks with earned badges for completed periods 🏅
- Keep a personal motivation scroll one click away 📜
- See milestone markers for the next big checkpoint 📍
- Spot when you're behind pace before it becomes a problem 🔴

## Highlights

- Weekly, monthly, yearly, and custom goals
- Per-deck goal groups plus custom goal windows
- Optional streak badges
- Optional milestone markers
- Optional motivation scroll popup with personal text
- Optional custom yearly start date
- One-bar-at-a-time layout for a cleaner home screen
- Compact design inspired by polished Anki dashboard add-ons

If you want Anki to feel a bit more goal-driven and a bit more motivating, this add-on is for you. ✨
