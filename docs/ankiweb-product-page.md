# Goal Tracking Progress Bar

This add-on adds configurable goal bars to Anki's home screen.

It can track:

- weekly goals
- monthly goals
- yearly goals
- custom goals

Goals can be created for individual deck groups or for broader study plans, and each goal can use one of these metrics:

- reviews completed
- new cards learned
- study time in minutes

The add-on includes:

- separate goal groups for different decks
- custom goals with configurable dates
- behind-pace indicator
- milestone markers
- streak badges
- optional Review Heatmap-inspired visual style

The widget is shown directly on the deck browser, so progress is visible without opening another screen.
